find_best_clf.get_score_for_10_most_common_classes(X_test, y_test, most_common_classes, clf)[source]¶

This function is responsible for measuring the prediction scores during the test phase

Parameters

  • X_test (numpy array) – the test set features

  • y_test (numpy array) – the test set class labels

  • most_common_classes (list) – the 10 most common (most populated) classes

  • clf (scikit-learn classifier object) – the classifier object that is used for the predictions

Returns

  • top_k_errors (list) – a list containing the top-k-error measurements

  • baseline_accuracy (float) – the baseline accuracy (the most populated class is assigned to every prediction)

  • baseline_f_score (float) – same as above, but f_score

  • accuracy_score(y_test, y_pred) (float) – the accuracy score as computed by scikit-learn

  • f1_score(y_test, y_pred, average=’weighted’) (float) – the weighted f1-score as computed by scikit-learn

  • f1_score(y_test, y_pred, average=’macro’) (float) – the macro f1-score as computed by scikit-learn
