textual_feature_extraction.get_features_top_k(ids, conn, args, k, test_ids=None)[source]¶

This function extracts frequent terms from the whole corpus of POI names. During this process, it optionally uses stemming. Selects the top-K most frequent terms and creates feature positions for each of these terms. It unifies the functionality of all the aforementioned top-k-feature-related function.

Parameters

  • ids (list) – the ids of the pois for which we want to this function to run

  • conn ((redundant)) –

  • args (dictionary) – several arguments that are needed for functionality purposes

  • k (float) – the percentage of the k most frequent terms discussed earlier

  • test_ids (list) – contains the ids of the test set pois for which we want this function to run, if no test ids it defaults to None

Returns

  • a dictionary that maps each poi to a boolean list that indicates whether

  • a term inside the k most frequent terms is found within that poi name.

textual_feature_extraction.get_poi_id_to_class_centroid_similarities(ids, poi_id_to_encoded_labels_dict, encoded_labels_set, conn, args, encoded_labels_corpus_dict, test=False)[source]¶

This function is responsible for mapping the pois to a list of size equal to the number of different classes in the dataset that contains the similarity between each poi’s textual elements to those of each class.

Parameters

  • ids (list) – the ids of the pois for which we want the similarity features to be extracted

  • poi_id_to_encoded_labels_dict (dictionary) – dictionary mapping each poi id to the encoded label of the class it belongs to

  • encoded_labels_set (set) – set containing the class label codes

  • conn ((redundant)) –

  • args (dictionary) – several arguments that are needed for functionality purposes

  • encoded_labels_corpus_dict (dictionary) – dictionary mapping each class label code to the corpus of the poi ids of that class

  • test (boolean) – boolean variable indicating whether we want these features to be extracted for members of the training set or not

Returns

  • poi_id_to_similarity_per_label (dictionary) – the dictionary discussed in the description

  • encoded_labels_corpus_dict (dictionary) – dictionary mapping each class label code to the corpus of the poi ids of that class
